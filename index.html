O que é DIDV?

DIDV significa Dispositivo de Interface para Deficientes Visuais. Consiste no desenvolvimento de um equipamento para leitura e escrita de textos através de interfaces Braille, autônomo, ou seja, sem a necessidade de um computador para sua utilização, baseado em licenças abertas de software e hardware.

leitura de txt ou livro digital ePub através de linha Braille

Sistema autônomo para composição de textos planos (*.TXT) e leitura de arquivos digitais, sejam textos planos (*.TXT) ou livros digitais (*.EPUB) por deficientes visuais através do Braille.

Composição de textos através de teclado braille

Na função “Escrever”, o DIDV permite ao usuário criar textos, usando o teclado Braille de seis teclas. As teclas são utilizadas para compor caracteres Braille, que são convertidas para texto comum e salvas.

ACESSO A ARQUIVOS VIA PENDRIVE

Para usar o DIDV é necessário utilizar um pendrive, pois nele os textos lidos e criados são armazenados. Através dele, também é possível compartilhar arquivos para leitura no DIDV ou criados nele para leitura em computadores.
